ESO Set Library

Server is up and running!

All this data has been gather from eso-sets.com thanks to Woeler.

ESO GQL API is located at /graphql

ESO REST API has 2 endpoints.

  • Get all sets: `/sets`
  • Get set by name: `/sets/:name` (Zaan)

Example:

{{set.name}}
Type: {{set.type}}
{{set.name}}
Type: {{set.type}}
